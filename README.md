# Dotfiles

this folder mappes to the home directory.

## Requirements

Minimum `stow` version should be [`2.4.0`](https://github.com/aspiers/stow/issues/33).

## Mapping the dotfiles

`stow --dotfiles <folder>`

## Delete the dotfiles

`stow --dotfiles -D <folder>`

## Refresh the dotfiles

`stow -R <folder>`
